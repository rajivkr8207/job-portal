## Installation

1. Clone the repository:
   ```bash
   git clone https://github.com/rajivkr8207/job-portal.git
   ```
2. Change into the project directory:
   ```bash
   cd job-portal
   ```
3. Install the dependencies:
   ```bash
   npm i
   ```
4. Run command
   ```bash 
   npm run dev
